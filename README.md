# 🎮 Quiz Area

Quiz Area is a modern, interactive quiz game built using **Python** and **Streamlit**. It provides an engaging gaming experience with a sleek UI, multiple quiz categories, dynamic scoring, and progress tracking.

## ✨ Features

- 🎯 Multiple quiz categories
  - General Knowledge
  - Science
  - Movies & Pop Culture
  - Sports
- 🎮 Gaming-inspired user interface
- 📊 Real-time score tracking
- 🔥 Streak bonus system
- ⚡ Speed bonus for quick answers
- 📈 Progress bar
- 🏆 High score tracking
- 🎵 Optional background music
- 🎨 Custom CSS animations and glassmorphism design
- 📱 Responsive Streamlit interface

## 🛠️ Technologies Used

- Python
- Streamlit
- HTML/CSS (Custom Styling)

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run quiz_game_app.py
```
